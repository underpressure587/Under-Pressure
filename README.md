# 🎮 Under Pressure

Under Pressure é um jogo de simulação estratégica baseado em decisões, onde o jogador gerencia uma empresa ao longo de várias rodadas.

---

# 🔄 Como o jogo funciona

O jogo acontece em **rodadas**.

Em cada rodada:

1. Uma situação é apresentada  
2. O jogador escolhe uma ação  
3. A escolha gera consequências  
4. Os indicadores são atualizados  
5. O jogo avança para a próxima rodada  

---

# 📊 Indicadores

Os indicadores representam o estado da empresa.

Exemplos:
- desempenho financeiro  
- eficiência  
- gestão de equipe  

Cada decisão pode:
- melhorar indicadores  
- piorar indicadores  
- gerar efeitos indiretos  

---

# 🎭 Eventos

Durante o jogo podem ocorrer eventos inesperados que:

- alteram indicadores  
- mudam decisões disponíveis  
- impactam o andamento da rodada  

---

# 🏢 Setores

O jogador pode atuar em diferentes setores, como:

- tecnologia  
- indústria  
- logística  

Cada setor possui características próprias.

---

# 🎯 Objetivo

Tomar decisões estratégicas ao longo das rodadas para manter os indicadores equilibrados e alcançar o melhor resultado possível ao final do jogo.
